# 异步函数

## 广义的异步函数

任何通过回调、事件、定时器等机制实现非阻塞操作的函数（例如 setTimeout、fs.readFile（Node.js）、fetch 等）。这类函数本身不一定返回 Promise，它们往往通过回调函数或事件来传递结果。



## 狭义的异步函数

特指使用 async 关键字声明的函数（即 async 函数）。这是 ES2017（ES8）引入的语法


### 返回值

1. 非 Promise 值 
   
  该值会被 Promise.resolve() 包装，成为一个 resolved 状态的 Promise。

  ```javascript
  async function foo() {
    return 42;
  }
  foo(); // 返回一个 Promise，状态为 resolved 值为 42
  foo().then(console.log); // 输出 42
  ```

2. Promise 值

  该 Promise 会直接作为函数返回的 Promise（即不会额外包装）

  ```javascript
  async function bar() {
    return Promise.resolve('hello');
  }
  bar(); // 返回一个 resolved 为 'hello' 的 Promise
  bar().then(console.log); // 输出 hello
  ```

3. 不写 return 语句

相当于返回 undefined，依然会被包装为 resolved 的 Promise。

  ```javascript
  async function baz() {
    // 没有 return
  }
  baz(); // 返回一个 resolved 值为 undefined 的 Promise
  baz().then(console.log); // 输出 undefined
  ```

4. 函数体内抛出错误   或  返回一个 rejected 的 Promise

  返回的 Promise 会被 reject，携带该错误。

  ```javascript
  async function qux() {
    throw new Error('出错了');
  }
  qux(); // 返回一个 rejected 的 Promise，错误为 Error('出错了')
  qux().catch(err => console.error(err.message)); // 输出 "出错了"
  ```
