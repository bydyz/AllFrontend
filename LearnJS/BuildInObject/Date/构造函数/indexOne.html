JavaScript Date对象详解

探索JavaScript中Date对象的各种构造函数和使用方法

Date对象概述

JavaScript的Date对象用于处理日期和时间。Date对象基于Unix时间戳，即自1970年1月1日（UTC）起经过的毫秒数。

注意：JavaScript中的月份是从0开始计数的（0 = 一月，1 = 二月，...，11 = 十二月）。

当前日期和时间

const now = new Date();
console.log(now); // 输出当前日期和时间

Date构造函数

new Date()

创建当前日期和时间的Date对象。

const now = new Date();

new Date(value)

使用时间戳（毫秒）创建Date对象。

const date = new Date(1643673600000);

new Date(dateString)

使用日期字符串创建Date对象。

const date = new Date("2022-02-01T12:00:00");

new Date(dateObject)

复制现有的Date对象。

const original = new Date();
const copy = new Date(original);

new Date(year, monthIndex)

使用年份和月份创建Date对象。

const date = new Date(2022, 1); // 2022年2月

new Date(year, monthIndex, day)

使用年份、月份和日期创建Date对象。

const date = new Date(2022, 1, 15); // 2022年2月15日

new Date(year, monthIndex, day, hours)

使用年份、月份、日期和小时创建Date对象。

const date = new Date(2022, 1, 15, 14); // 2022年2月15日14:00

new Date(year, monthIndex, day, hours, minutes, seconds, milliseconds)

使用所有时间组件创建Date对象。

const date = new Date(2022, 1, 15, 14, 30, 45, 500);

Date()函数

直接调用Date()函数（不带new关键字）会返回当前日期和时间的字符串表示，而不是Date对象。

const dateString = Date(); // 返回字符串
const dateObject = new Date(); // 返回Date对象

注意：直接调用Date()（不带new）返回的是字符串而不是Date对象，因此不能使用Date对象的方法。

常用的Date方法

方法 描述 示例
getFullYear() 获取年份（四位数） date.getFullYear()
getMonth() 获取月份（0-11） date.getMonth()
getDate() 获取日期（1-31） date.getDate()
getDay() 获取星期几（0-6） date.getDay()
getHours() 获取小时（0-23） date.getHours()
getMinutes() 获取分钟（0-59） date.getMinutes()
getSeconds() 获取秒数（0-59） date.getSeconds()
getMilliseconds() 获取毫秒（0-999） date.getMilliseconds()
getTime() 获取时间戳（毫秒） date.getTime()
toDateString() 返回日期部分的字符串 date.toDateString()
toTimeString() 返回时间部分的字符串 date.toTimeString()
toISOString() 返回ISO格式的字符串 date.toISOString()

Date方法示例

const now = new Date();
const year = now.getFullYear();
const month = now.getMonth(); // 0-11
