# 仅是迭代器（仅实现了迭代器协议的对象）

* 对象必须包含一个 next() 方法。
* next() 返回一个对象，包含两个属性：
  * value：当前产生的值（可选，如果 done: true 可省略）。
  * done：布尔值，true 表示迭代结束，false 表示还有更多值。


不能直接用于 for...of，但可以通过手动调用 next() 遍历。



# 可迭代对象

* 对象必须包含一个 `[Symbol.iterator]` 方法。
* 该方法返回一个迭代器对象（通常是 this 或一个新的迭代器）。
* 当使用 for...of、扩展运算符 ...、Array.from() 等语法时，JavaScript 会自动调用这个方法获取迭代器。
