CSS Grid布局详解

了解如何使用grid-template-columns和grid-template-rows属性创建网格布局，并使用命名的网格线定位元素。

CSS代码示例

.grid-container {
  display: grid;
  /* 定义列：从[start]开始，一列自动宽度，一列1fr比例，到[end]结束 */
  grid-template-columns: [start] auto [main] 1fr [end];
  /* 定义行：header行80px，content行自动高度，footer行60px */
  grid-template-rows: [header] 80px [content] auto [footer] 60px;
  gap: 10px;
}
Header (grid-column: start / end; grid-row: header)
Sidebar (grid-column: start / main; grid-row: content)
Main Content (grid-column: main / end; grid-row: content)
Footer (grid-column: start / end; grid-row: footer)

代码详解

grid-template-columns: [start] auto [main] 1fr [end]; 这行代码定义了网格的列结构：

  • 创建了两列布局
  • 第一列的宽度为auto（根据内容自动调整）
  • 第二列的宽度为1fr（占用剩余可用空间）
  • 定义了三条垂直网格线：start, main 和 end

grid-template-rows: [header] 80px [content] auto [footer] 60px; 这行代码定义了网格的行结构：

  • 创建了三行布局
  • 第一行高度为80px
  • 第二行高度为auto（根据内容自动调整）
  • 第三行高度为60px
  • 定义了四条水平网格线：header, content 和 footer

通过使用这些命名的网格线，我们可以轻松地将网格项目放置到网格中的特定区域，例如：

  • Header: 从start到end（跨越所有列），位于header行
  • Sidebar: 从start到main（第一列），位于content行
  • Main Content: 从main到end（第二列），位于content行
  • Footer: 从start到end（跨越所有列），位于footer行
