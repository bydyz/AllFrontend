CSS Grid - grid-column与grid-row详解

了解如何使用grid-column和grid-row属性在网格布局中精确定位元素。

grid-column与grid-row属性

grid-column 和 grid-row 属性用于指定网格项目在网格布局中的位置和跨度。

这些属性可以使用以下值：

  • 线编号：例如 grid-column: 1 / 3（从第1条线开始到第3条线结束）
  • 命名线：例如 grid-column: main-start / main-end
  • span关键字：例如 grid-column: 1 / span 2（从第1条线开始，跨越2列）

注意：网格线编号从1开始，而不是0。每条垂直线定义列的边界，每条水平线定义行的边界。

基础网格示例代码

.grid-container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-template-rows: repeat(3, 100px);
  gap: 10px;
}

.item-1 {
  grid-column: 1 / 3; /* 从第1条线开始到第3条线结束 */
  grid-row: 1; /* 在第1行 */
}

.item-2 {
  grid-column: 3 / 5; /* 从第3条线开始到第5条线结束 */
  grid-row: 1 / 3; /* 从第1条线开始到第3条线结束 */
}

基础网格示例

grid-column: 1 / 3;
grid-row: 1;
grid-column: 3 / 5;
grid-row: 1 / 3;
grid-column: 1 / 2;
grid-row: 2 / 4;
grid-column: 2 / 4;
grid-row: 3;
grid-column: 4;
grid-row: 3;

命名网格线示例代码

.named-grid {
  display: grid;
  grid-template-columns: [start] 1fr [main-start] 2fr [main-end] 1fr [end];
  grid-template-rows: [header] 100px [content] 150px [footer] 80px;
  gap: 10px;
}

.named-item-1 {
  grid-column: start / end; /* 从start线到end线 */
  grid-row: header; /* 在header行 */
}

.named-item-2 {
  grid-column: main-start / main-end; /* 从main-start线到main-end线 */
  grid-row: content; /* 在content行 */
}

命名网格线示例

grid-column: start / end;
grid-row: header;
grid-column: main-start / main-end;
grid-row: content;
grid-column: start;
grid-row: footer;
grid-column: main-start / end;
grid-row: footer;

属性值详解

grid-column属性

grid-column 是以下两个属性的简写：

  • grid-column-start：定义项目开始的垂直网格线
  • grid-column-end：定义项目结束的垂直网格线

grid-row属性

grid-row 是以下两个属性的简写：

  • grid-row-start：定义项目开始的水平网格线
  • grid-row-end：定义项目结束的水平网格线

使用span关键字

可以使用span关键字指定项目应该跨越的轨道数量：

  • grid-column: 2 / span 3：从第2条线开始，跨越3列
  • grid-row: span 2：自动跨越2行
  • grid-column: span 2：自动跨越2列

提示：如果只指定一个值，第二个值将设置为auto，项目默认跨越一个轨道。
