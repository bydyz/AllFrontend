CSS Grid: justify-items 与 align-items

详解justify-items和align-items属性的区别、用法和应用场景

justify-items 与 align-items 概述

在CSS Grid布局中，justify-items和align-items属性用于控制网格项目在网格单元格内的对齐方式。

注意：这些属性应用于网格容器，并影响所有网格项目的对齐方式。

justify-items

控制网格项目在水平方向（行轴）上的对齐方式

.grid-container {
  justify-items: center;
}

align-items

控制网格项目在垂直方向（列轴）上的对齐方式

.grid-container {
  align-items: center;
}
justify-items (行轴)
align-items (列轴)

属性值详解

属性值 描述 justify-items效果 align-items效果
start 项目对齐到单元格的起始边缘 左对齐 上对齐
end 项目对齐到单元格的结束边缘 右对齐 下对齐
center 项目在单元格中居中对齐 水平居中 垂直居中
stretch 项目拉伸以填充整个单元格（默认值） 水平拉伸 垂直拉伸

}

注意事项

提示：可以使用justify-self和align-self属性对单个网格项目进行对齐设置，这会覆盖容器的justify-items和align-items设置。

.special-item {
  justify-self: end; /* 单独设置水平对齐 */
  align-self: start; /* 单独设置垂直对齐 */
}
