与 <pre> 标签等同的 CSS 样式

学习如何使用CSS实现类似<pre>标签的效果，保持文本格式和空白符

<pre> 标签的作用

<pre> (Preformatted Text) 标签用于定义预格式化的文本。其中的文本会保留空格和换行符，并以等宽字体显示。

普通文本

这是一段普通文本， 它 会 忽略 多余的空格 和 换行。

<pre> 文本

这是一段预格式化文本，
它   会   保留   所有空格
和
换行。

注意：<pre> 标签默认样式包括：display: block;, font-family: monospace;, white-space: pre;, margin: 1em 0;

实现 pre 效果的关键 CSS 属性

属性 描述 值
white-space 控制空白符的处理方式 pre, pre-wrap, pre-line
font-family 指定字体系列 monospace, 'Courier New', monospace 等
overflow-x 控制水平溢出内容 auto, scroll
tab-size 指定制表符的宽度 数字（如4）
display 指定显示方式 block, inline-block

white-space 属性详解

white-space: pre

function example() {
console.log("Hello, World!");
return true;
}

保留所有空白符，不自动换行

white-space: pre-wrap

function example() { console.log("Hello, World!"); return true; }

保留所有空白符，但自动换行

white-space: pre-line

function example() { console.log("Hello, World!"); return true; }

合并空格，保留换行符，自动换行

white-space: normal

function example() { console.log("Hello, World!"); return true; }

忽略多余空白符，自动换行（默认行为）

创建类似 <pre> 的样式

/* 基本 pre 样式 */
.pre-style {
  font-family: monospace;
  white-space: pre;
  display: block;
  overflow-x: auto;
  background-color: #f4f4f4;
  padding: 1em;
  border-radius: 4px;
  margin: 1em 0;
}

应用示例

function calculate(a, b) { // 这是一个示例函数 const result = a + b; /* * 多行注释 * 也会保留格式 */ return result; }

高级 pre 样式实现

/* 高级 pre 样式 - 类似代码编辑器 */
.pre-style-code {
  font-family: 'Fira Code', monospace;
  background: linear-gradient(135deg, #2d3436 0%, #1e272e 100%);
  color: #f5f7fa;
  padding: 20px;
  border-radius: 5px;
  overflow-x: auto;
  line-height: 1.5;
  tab-size: 4;
  white-space: pre;
  margin: 15px 0;
}
