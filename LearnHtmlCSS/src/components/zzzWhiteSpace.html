CSS white-space 属性详解

了解white-space属性如何控制空白符的处理方式

white-space 属性概述

white-space CSS属性用于设置元素内部空白的处理方式，控制文本中的空格、制表符和换行符的显示方式。

空白符类型：

  • 空格（Space）
  • 制表符（Tab）
  • 换行符（Newline）

空格 ·

在HTML中表示为 空格 或 &nbsp;

换行符 ↵

在HTML中表示为 换行 或 <br>

制表符 →

在HTML中表示为 制表符 或

white-space 属性值

属性值 空格处理 换行处理 自动换行 文本方向

nowrap 合并 忽略 否 从左到右

pre-line 合并 保留 是 从左到右

white-space: normal

默认值。合并连续的空白符为一个空白符，忽略换行符，文本自动换行。

.normal-example {
  white-space: normal;
}
normal 这是一段 包含 多个空格 的文本。 它还包含 换行符和 制表符。 文本会自动换行以适应容器宽度。

特点：

  • 合并连续的空白符为一个空白符
  • 忽略换行符
  • 文本自动换行
  • HTML的默认行为

white-space: nowrap

合并连续的空白符，忽略换行符，文本不换行。

.nowrap-example {
  white-space: nowrap;
}
nowrap 这是一段 包含 多个空格 的文本。 它还包含 换行符和 制表符。 文本不会自动换行，会一直延伸直到遇到<br>标签。

特点：

  • 合并连续的空白符
  • 忽略换行符
  • 文本不换行
  • 常用于创建水平滚动或单行文本

white-space: pre

保留所有空白符（像<pre>标签一样），文本不自动换行。

.pre-example {
  white-space: pre;
}
pre 这是一段 包含 多个空格 的文本。 它还包含 换行符和 制表符。 文本保留所有空白符，但不自动换行。

特点：

  • 保留所有空白符
  • 保留换行符
  • 文本不自动换行
  • 行为类似于<pre>标签

white-space: pre-wrap

保留所有空白符，文本自动换行。

.pre-wrap-example {
  white-space: pre-wrap;
}
pre-wrap 这是一段 包含 多个空格 的文本。 它还包含 换行符和 制表符。 文本保留所有空白符，并且会自动换行以适应容器宽度。

特点：

  • 保留所有空白符
  • 保留换行符
  • 文本自动换行
  • 非常适合显示代码

white-space: pre-line

合并连续的空白符，但保留换行符，文本自动换行。

.pre-line-example {
  white-space: pre-line;
}
pre-line 这是一段 包含 多个空格 的文本。 它还包含 换行符和   制表符。 文本合并连续空格但保留换行符，并且会自动换行。

特点：

  • 合并连续的空白符
  • 保留换行符
  • 文本自动换行
  • 适用于保留段落结构但不需要保留空格的情况

white-space: break-spaces

保留所有空白符，文本自动换行，空格处也可以换行。

.break-spaces-example {
  white-space: break-spaces;
}
这是一段 包含 多个空格 的文本。 它还包含 换行符和 制表符。 文本保留所有空白符，并且会在空格处换行以适应容器宽度。

特点：

  • 保留所有空白符
  • 保留换行符
  • 文本自动换行，包括在空格处
  • 与pre-wrap类似，但在空格处的换行行为不同
