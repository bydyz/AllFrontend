CSS text-transform 属性详解

该属性具有继承性，子元素会继承父元素的设置

text-transform 属性示例

none (默认值)
原始文本：Hello World! This is a MIXED case Example.

不进行任何转换，保持文本原始状态。

capitalize
原始文本：hello world! this is a MIXED case example.

将每个单词的第一个字母转换为大写。

uppercase
原始文本：Hello World! This is a MIXED case Example.

将所有字母转换为大写。

lowercase
原始文本：Hello World! This is a MIXED case Example.

将所有字母转换为小写。

full-width
原始文本：Hello World! 123 ABC.

将字符转换为全角形式，主要用于中日韩文字排版。

属性总结

none

默认值，不进行任何文本转换，保持原始状态。

capitalize

将每个单词的首字母转换为大写，其他字母保持不变。

uppercase

将所有字母转换为大写形式。

lowercase

将所有字母转换为小写形式。

full-width

将字符转换为全角形式，主要用于中日韩文字排版。

实际应用场景

品牌一致性

确保标题或导航文本始终以特定大小写显示，即使内容输入不一致。

表格数据

统一表格中某一列文本的格式，例如将所有国家名称首字母大写。

按钮设计

按钮文本通常使用全大写形式，使用 text-transform 可以轻松实现。

多语言排版

使用 full-width 值可以改善拉丁字符与中日韩文字的混合排版效果。
