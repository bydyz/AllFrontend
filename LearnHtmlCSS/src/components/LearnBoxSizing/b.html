说明：

border-box: 设置的宽度就是实际占用的空间，包含 padding 和 border

content-box: 设置的宽度只是内容宽度，实际占用空间 = 内容 + padding + border

这就是为什么现代 CSS 框架都使用 box-sizing: border-box
