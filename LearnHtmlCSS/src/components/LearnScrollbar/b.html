::-webkit-scrollbar 伪类详解

自定义 WebKit 浏览器滚动条的完整指南

基本介绍

::-webkit-scrollbar 是 WebKit 浏览器（Chrome、Safari、Edge等）特有的 CSS 伪类，用于自定义滚动条样式。

注意： 这是非标准属性，仅在 WebKit 内核的浏览器中有效。Firefox 使用 scrollbar-width 和 scrollbar-color。

兼容性

支持的浏览器：

  • Google Chrome ✅
  • Safari ✅
  • Microsoft Edge ✅
  • Opera ✅
  • Firefox ❌ (使用标准属性)

基本结构

WebKit 滚动条由多个部分组成：

  • ::-webkit-scrollbar - 整个滚动条
  • ::-webkit-scrollbar-track - 滚动条轨道
  • ::-webkit-scrollbar-thumb - 滚动条滑块
  • ::-webkit-scrollbar-button - 滚动条按钮
  • ::-webkit-scrollbar-track-piece - 轨道部分
  • ::-webkit-scrollbar-corner - 滚动条角落

可设置的属性

属性 适用伪类 描述 示例值
width ::-webkit-scrollbar 滚动条宽度 8px, 12px, thin
height ::-webkit-scrollbar 滚动条高度（水平滚动条） 8px, 12px
background 所有部分 背景颜色或渐变 #f0f0f0, linear-gradient()
background-color 所有部分 背景颜色 red, #ff0000, rgba()
border 所有部分 边框样式 1px solid #ccc
border-radius thumb, track 圆角边框 4px, 10px
box-shadow thumb, track 阴影效果 2px 2px 5px rgba(0,0,0,0.3)
display 所有部分 显示/隐藏 none, block

详细示例

/* 示例1: 渐变滚动条 */ #demoScroll::-webkit-scrollbar { width: 16px; } #demoScroll::-webkit-scrollbar-track { background: linear-gradient(180deg, #667eea, #764ba2); border-radius: 10px; } #demoScroll::-webkit-scrollbar-thumb { background: linear-gradient(180deg, #f093fb, #f5576c); border-radius: 10px; border: 2px solid white; } #demoScroll::-webkit-scrollbar-thumb:hover { background: linear-gradient(180deg, #f5576c, #f093fb); } #demoScroll::-webkit-scrollbar-button { background: #3498db; border-radius: 50%; }
/* 示例2: 迷你滚动条 */ #demoScroll::-webkit-scrollbar { width: 6px; } #demoScroll::-webkit-scrollbar-track { background: transparent; } #demoScroll::-webkit-scrollbar-thumb { background: #e74c3c; border-radius: 3px; } #demoScroll::-webkit-scrollbar-thumb:hover { background: #c0392b; } /* 隐藏滚动条按钮 */ #demoScroll::-webkit-scrollbar-button { display: none; }

高级技巧

1. 隐藏但保留功能

/* 隐藏滚动条但保留滚动功能 */ .container::-webkit-scrollbar { width: 0px; background: transparent; } /* 或者完全隐藏 */ .container { scrollbar-width: none; /* Firefox */ -ms-overflow-style: none; /* IE/Edge */ } .container::-webkit-scrollbar { display: none; }

2. 不同状态的样式

/* 正常状态 */ ::-webkit-scrollbar-thumb { background: #bdc3c7; } /* 悬停状态 */ ::-webkit-scrollbar-thumb:hover { background: #95a5a6; } /* 激活状态（正在拖动） */ ::-webkit-scrollbar-thumb:active { background: #7f8c8d; } /* 禁用状态 */ ::-webkit-scrollbar-thumb:disabled { background: #ecf0f1; }

浏览器兼容性解决方案

/* WebKit 浏览器 (Chrome, Safari, Edge) */ .scrollable-element::-webkit-scrollbar { width: 12px; } .scrollable-element::-webkit-scrollbar-track { background: #f1f1f1; } .scrollable-element::-webkit-scrollbar-thumb { background: #888; border-radius: 6px; } .scrollable-element::-webkit-scrollbar-thumb:hover { background: #555; } /* Firefox */ .scrollable-element { scrollbar-width: thin; scrollbar-color: #888 #f1f1f1; } /* IE/Edge 旧版本 */ .scrollable-element { -ms-overflow-style: -ms-autohiding-scrollbar; }
