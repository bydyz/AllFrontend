### Vue.js 是一个现代的前端框架，它采用了 MVVM（Model-View-ViewModel）架构模式
* Model：表示应用程序的数据层，通常是来自数据库或其他持久化存储的数据。
  * 在 Vue.js 应用中，Model 通常指的是组件中的 data 或 props，它们代表了应用的状态或数据。

* View：表示用户界面，即用户看到并与之交互的部分。
  * View 在 Vue.js 中是指组件的模板部分，通常是 .vue 文件中的 template 标签内容。

* ViewModel：作为中介层，负责处理业务逻辑，并将 Model 的数据转换为适合 View 显示的形式，同时监听 View 上的用户输入并将这些变化反馈给 Model。
  * 在 Vue.js 中，ViewModel 主要由 Vue 实例本身以及相关的响应式系统组成。Vue 实例充当 ViewModel 的角色，负责处理数据绑定、事件监听、生命周期管理等。

<br/>

- 在 Vue.js 中，数据绑定和视图更新是通过双向数据绑定机制实现的。这意味着：
  - 当 data 或 props 发生变化时，视图会自动更新。
  - 当用户通过表单元素修改数据时，data 也会自动更新。
