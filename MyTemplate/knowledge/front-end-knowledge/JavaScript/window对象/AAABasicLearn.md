在JavaScript中，window 对象是浏览器环境中的全局对象，代表当前浏览器窗口或标签页。它不仅充当了全局命名空间，还提供了与浏览器窗口交互的方法和属性。

<br/>

### window 对象的主要功能
1. 全局作用域：
   * 在浏览器环境中，所有全局变量和函数实际上是 window 对象的属性和方法。例如，定义一个全局变量 var x = 10; 实际上等同于 window.x = 10;
   * 同样，定义一个全局函数 function foo() {} 实际上是 window.foo = function() {};。
2. 浏览器窗口操作：
   * window 对象提供了许多用于控制浏览器窗口的方法和属性，如调整窗口大小、移动窗口位置、打开新窗口等。
3. 定时器：
   * setTimeout() 和 setInterval() 方法用于设置一次性定时器和重复定时器，分别在指定的时间后执行一次代码或每隔一段时间执行一次代码。
4. 导航和历史管理：
   * window.location 对象包含了当前页面的URL信息，并提供了导航到其他页面的方法。
   * window.history 对象允许你访问浏览器的历史记录栈，支持前进和后退操作。
5. 弹出对话框：
   * alert()、confirm() 和 prompt() 方法用于显示简单的用户界面对话框。
6. 事件处理：
   * window 对象可以用来监听和触发各种事件，如页面加载完成 (load)、窗口大小变化 (resize) 等。
